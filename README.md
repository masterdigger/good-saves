# Good Saves

En modul för att hantera och skicka formulärdata med dynamiska värden.

## Installation

```bash
git clone https://github.com/masterdigger/good-saves.git
cd good-saves
poetry install
```

## Användning

```python
from good_saves.main import main_function

main_function()
```
